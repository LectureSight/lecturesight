\bundlefile{lecturesight-decorator-color.jar}

The bundle provides a service that implements an \texttt{ObjectDecorator}. For object that fit configurable size criteria, the service creates and updates color histograms. The number of bins of the histograms can be configured. 

\configproperties

\property{cv.lecturesight.decorator.color.width.min}{23}{Minimum width for an object to be anaylsed by this decorator.}

\property{cv.lecturesight.decorator.color.width.max}{48}{Maximum width for an object to be anaylsed by this decorator.}

\property{cv.lecturesight.decorator.color.height.min}{50}{Minimum height for an object to be anaylsed by this decorator.}

\property{cv.lecturesight.decorator.color.height.max}{80}{Maximum height for an object to be anaylsed by this decorator.}

\property{cv.lecturesight.decorator.color.channel.number}{4}{Number of bins for the histogram.}
